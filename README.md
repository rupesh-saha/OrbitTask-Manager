To-Do List Manager 

A sleek, responsive task management web application built with Vanilla JavaScript, Tailwind CSS, and daisyUI. This project focuses on providing a distraction-free environment for organizing daily agendas with persistent data storage.



✨ Features
Persistent Storage: Uses LocalStorage to ensure your tasks remain saved even after a browser refresh or restart.

Real-time Statistics: Automatically tracks "remaining" and "completed" task counts to help you stay on top of your goals.

Tabbed Filtering: Easily switch between viewing "All Tasks" and "Completed Tasks" for better organization.

Unique Task IDs: Implements Date.now() logic to ensure every task is unique, preventing deletion and completion bugs.

Modern UI: Built with daisyUI v5 and Tailwind CSS v4 for a clean, professional look with soft shadows and a Geist typography system.

Responsive Design: Fully optimized for both desktop and mobile viewing.




🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Styling: Tailwind CSS v4 & daisyUI v5

Icons: Font Awesome 7

Typography: Geist Sans via Google Fonts
